# Icon Files

The icons directory should contain PNG images for the extension:

- `icon16.png` - 16x16 pixels
- `icon48.png` - 48x48 pixels  
- `icon128.png` - 128x128 pixels

## Creating Icons

You can use the `icon128.svg` file as a template and export to PNG at the required sizes, or create custom icons.

## Temporary Solution

For development testing, you can:
1. Use any placeholder PNG images at the correct sizes
2. Or convert icon128.svg to PNG using an online tool or image editor
3. Chrome will display a generic icon if files are missing (extension will still work)

## Recommended Icon Design

- Simple, recognizable symbol
- Works at small sizes (16x16)
- Represents recording/tracking functionality
- Uses brand colors (#0066cc blue, #ff0000 red for recording dot)
